# ECMAScript-6-Cheat-Sheet
Summary of ECMAScript 6 features

## arrow functions
```javascript
    x => x + 1
    (x, y) => x + y
    x => {return x + 1}
```

```javascript
    Number.isFinite

    Number.MIN_SAFE_INTEGER // -9007199254740991
    Number.MAX_SAFE_INTEGER // 9007199254740991
```
